**🏦 Bank Customer Churn Analysis**
📌 Project Overview
    This project focuses on analyzing bank customer churn behavior using data analytics tools to identify key factors affecting customer attrition.
    The objective was to understand why customers leave the bank and provide data-driven recommendations to improve retention strategies.
    The analysis was performed on a dataset containing 10,000+ customer records.

**🎯 Problem Statement**
  Customer churn directly impacts revenue and long-term growth.
  The goal of this project was to:
  Identify key churn drivers
  Analyze customer behavior patterns
  Track retention-related KPIs
  Provide actionable business recommendations

**🛠️ Tools & Technologies Used**
  Excel – Data cleaning and preprocessing
  SQL – Data querying and churn analysis
  Power BI – Dashboard creation and KPI visualization

**🔎 Data Preparation**
    Cleaned raw data using Excel by handling missing values and removing duplicates.
    Standardized data formats to ensure consistency.
    Prepared structured datasets for analysis.

**📊 Analysis Approach**
    Used SQL joins, filters, and aggregations to analyze customer attributes.
    Studied relationships between churn and:
    Customer tenure
    Account balance
    Product usage
    Demographics
    Identified 3 major churn indicators contributing to a significant portion of customer attrition.

**📈 Dashboard & KPIs**
    Built interactive Power BI dashboards to track:
    Churn Rate
    Retention Trends
    Customer Segmentation
    Tenure vs Churn Analysis
    Product Usage Impact
    The dashboard enables quick insights into high-risk customer segments.

**💡 Key Insights**
    Customers with lower tenure showed higher churn probability.
    Certain product usage patterns were strongly linked to churn behavior.
    Specific customer segments required targeted retention strategies.

**🚀 Business Impact**
    Provided actionable recommendations to reduce churn.
    Highlighted high-risk customer segments for proactive engagement.\
    Demonstrated how data analysis can support customer retention strategies.

**🔮 Future Improvements**
    Implement predictive churn modeling using ML
    Add customer lifetime value analysis
    Deploy real-time churn monitoring dashboard

**🎯 Conclusion**
    This project demonstrates how structured data analysis using Excel, SQL, and Power BI can uncover churn drivers and support strategic decision-making.
    It combines data cleaning, SQL analysis, and business-focused visualization to solve a real-world banking problem.
